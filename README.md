# DateSurfer.Core – Clean Architecture Showcase

## 🎯 Live Demo

👉 [https://datesurfer-core-app-bzhbdkfgchb2ene3.germanywestcentral-01.azurewebsites.net](https://datesurfer-core-app-bzhbdkfgchb2ene3.germanywestcentral-01.azurewebsites.net)

## 🛠️ Technologie-Stack

| Component | Technology |
|-----------|------------|
| Backend | ASP.NET Core 8 |
| Architecture | Clean Architecture |
| Database | SQL Server / Azure SQL |
| ORM | Entity Framework Core |
| Testing | xUnit, Moq, FluentAssertions |
| Hosting | Azure App Service |

## 🏗️ Clean Architecture Struktur

- **Domain** - Entities, Enums, Interfaces
- **Application** - Use Cases, Handlers, Services  
- **Infrastructure** - Data Access, Repositories
- **Web** - Controllers, Views, UI

## 📊 Business Logic

Regelbasierte Gebührenberechnung für Mitgliedschaften:
- User mit Land, Alter und Username
- FeeRules definieren monatliche Gebühren
- Perfekt übertragbar auf Tax- und Royalty-Systeme
